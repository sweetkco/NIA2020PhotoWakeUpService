Установка / Blender

  • Предпосылки
  • Загрузка Verge3D
  • Установка для Windows
  • Установка для macOS или Linux

#

Предпосылки

Verge3D может работать поверх Blender, пакета 3D-моделирования и анимации с открытым исходным кодом, который необходимо установить первым (мы поддерживаем версии 2.80–2.83). Вы можете бесплатно загрузить Blender с официального сайта .

#

Загрузка Verge3D

Загрузите Verge3D для Blender для своей операционной системы (установщик .exe для Windows или архив XZ для macOS/Linux).

#

Установка для Windows

Дважды щелкните файл .exe , чтобы запустить установщик.

В Win 10 существует известная проблема, когда Windows Defender может отказать в запуске установщика (мы работаем над этим). Если вы видите синее окно с предупреждением, как на картинке ниже, нажмите ссылку More info:

и затем нажмите кнопку Run anyway:

Выберите папку для установки (или лучше оставьте ее по умолчанию). Убедитесь, что у вас достаточно разрешений на чтение и запись для этой папки. Установка в системный каталог, такой как «Program Files» или «Desktop», не рекомендуется.

После завершения установки вы можете открыть Blender. Если он у вас запущен, обязательно закройте и снова откройте.

Откройте окно Preferences и перейдите на вкладку Add-ons. Найдите и включите надстройку Verge3D, затем нажмите Save Preferences и закройте окно.

Начиная с версии 2.17, вы также можете запустить менеджер приложений без запуска Blender, щелкнув значок на рабочем столе или используя меню «Пуск».

#

Установка для macOS или Linux

Распакуйте архив XZ в каталог по вашему выбору. Убедитесь, что у вас достаточно разрешений на чтение и запись в этой папке. Не рекомендуется устанавливать в системный каталог.

В Blender вызовите окно Preferences, выбрав соответствующую опцию в меню Edit.

На вкладке File Paths найдите поле Scripts и укажите в нем папку, в которой установлен Verge3D. Закройте окно Preferences и перезапустите Blender .

Снова откройте окно Preferences и перейдите на вкладку Add-ons. Найдите и включите надстройку Verge3D, затем нажмите Save Preferences и закройте окно.

Установка завершена!
